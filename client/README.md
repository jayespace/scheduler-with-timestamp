- useEffect hook으로 현재 시간과 스케줄 업데이트 기능을 구현 하였습니다.
- component 구현으로 코드 재사용을 하였습니다.
- 메모 수정 기능은 modal로 만들어 간편하게 사용 할 수 있습니다.
- media query를 적용하여 모바일에서도 불편함 없이 사용할 수 있습니다.
